### HW35. EC2 — Account
``` 
В настройках аккаунта для EC2:
Включить шифрование EBS томов.
Для всех burstable инстансов выключить использование creditов. 
Включить использование EC2 Serial Console.
Включить все экспериментальный опции.
```
